# JobLinker
  <p>
    This is a job search website template. The website provides login and 
    logout functions. After logging in, the website displays user profiles, 
    community information, and people you may know. Users can search for job 
    information posted by various companies. Users can view information in v
    arious modules, such as: notifications, work information, text messages, 
    schedules, etc. Users can also post job search information to the 
    ommunity, add people you may know, and expand your professional network.
  </p>
